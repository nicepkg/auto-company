# Security Program Summary

Administrative and production systems require phishing-resistant multi-factor authentication using FIDO2 security keys.
Privileged access to cloud administration consoles is restricted to role-based groups and reviewed quarterly.
Customer data is encrypted at rest with AES-256 and encrypted in transit using TLS 1.2 or higher.
Security operations center analysts triage high-severity alerts within 30 minutes under the incident severity matrix.
